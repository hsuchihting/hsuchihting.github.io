Chart.js 簡介

Chart.js 是一款彈性很高的圖表 JavaScript library，支援八種常見的統計圖表類型，夠將圖表混合在一起使用，也支援動畫的效果，讓我們製作出來的圖表更加精美！Chart.js 可以與 ES6 模塊，純 JavaScript 和模塊加載器一起使用。

載入 chart.js

您可以從 GitHub 版本下載最新版本的 Chart.js 或使用 Chart.js CDN。
