EJS 模板語法介紹

EJS 簡介

EJS 是一個簡易的模板語法，又有高校的嵌入式 JavaScript 模板引擎，有點類似 ES6 的模板語法，只是這個是使用 ejs 的檔案格式中去撰寫 HTML 標籤，並且把一些迴圈的語法寫在模板上。
