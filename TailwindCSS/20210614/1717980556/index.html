智能提示的方便之處

順便提一下智能提示有一個很方便的地方，也就是當我在打想要使用的 class 時，旁邊會跳出提示訊息，告訴我此 class 在編譯成 CSS 後會是什麼設定，這樣就不用一直去測試 Tailwind 的 class 效果是什麼囉！
