下方宣告一個變數 a 跟一個屬性 b ，並給予值。

打開開發人員工具輸入 window 物件，可以看到有新增這兩筆物件資料。

透過JS 筆記 - Object 物件的基本認識知道新增與刪除的方法。

透過一元運算子來刪除這兩筆資料，並輸入 window 檢視，

發現 b 不見了，a 還在。

如果不刪除 a 跟 b，透過 console 檢視的話，會看到此狀態。
