JavaScript 概述

一般來說，完整的 JavaScript 包括以下幾個部分：

  • ECMAScript，描述了該語言的語法和基本物件
  • 文件物件模型（DOM），描述處理網頁內容的方法和介面
  • 瀏覽器物件模型（BOM），描述與瀏覽器進行互動的方法和介面

JavaScript 的基本特點如下：

  • 是一種解釋性程式語言（代碼不進行預編譯）。
  • 主要用來向 HTML 頁面添加互動行為。
  • 可以直接嵌入 HTML 頁面，但寫成單獨的 JS 檔案有利於結構和行為的分離。

JavaScript 常用來完成以下任務：

  • 嵌入動態文字於 HTML 頁面
  • 對瀏覽器事件作出回應
  • 讀寫 HTML 元素
  • 在資料被提交到伺服器之前驗證資料
  • 檢測訪客的瀏覽器資訊
  • 控制 cookies，包括建立和修改等
