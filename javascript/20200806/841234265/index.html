Truthy 真值

下方有一個 if 判斷式，若判斷內容為 True 就會執行 console 內容。

若改成這樣也是會正確執行。

原因是因為 if 判斷內容為真值。

falsy

但如果將判斷的內容改為空值或 undefined 或 0，屬於 falsy，就會執行 else。

但如果引號中有任何值(包含空白)，就會變成 truthy。

上方判斷中插入空白，就會執行 Truthy 內容。
