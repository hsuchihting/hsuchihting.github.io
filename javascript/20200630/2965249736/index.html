小提示：

在 JS 中引號的用法很重要，不能混搭使用，以這行為例：
'<h1 class="blue">innerHTML HELLO</h1>'，
倘若改成都是單引號：
'<h1 class='blue'>innerHTML HELLO</h1>'，
電腦會誤認為：
'<h1 class=' + blue + '>innerHTML HELLO</h1>'

所以使用 JS 的引號一定要一致!
