心得

透過 change 的 function 內容的建立，執行的順序要一個口令一個動作，拆解流程就會比較好思考。

  • 自己整理的步驟
  1. 點選自己的選項的值。
  2. 選到自己後，要組字串給清單用。(資料來自於 for)
  3. for 裡面馬上接判斷式，給予要選擇的條件。
  4. 在條件內建立要顯示的字串。
  5. 並在 for 迴圈外，將要顯示的變數內容，透過 innerHTML = str(資料就會丟回空字串中)。
