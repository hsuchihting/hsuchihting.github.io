發表於 2020-07-28 分類於 JavaScript Disqus：
JS 的事件有哪些?

瀏覽器中有許多的事件，筆記常用的像是滑鼠點擊、移動事件、鍵盤輸入事件、冒泡事件…等，事件通常與函式搭配使用，在事件發生前函式不會被執行！
