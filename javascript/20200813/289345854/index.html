簡介箭頭函式與傳統函式的差異點

箭頭函式沒有 arguments

arguments 是類陣列，跟一般陣列不同，但在查驗結果時會是用陣列的型態顯示資料，下方有一函式，在傳統函式會自動新增 arguments 這個參數，會將參數的值帶入其中。

箭頭函式的 this

但在箭頭函式就有些許不同，箭頭函式沒有自己的 this。

將下方 setTimeout 的函式改成箭頭函式後，會發現箭頭函式的 this 會向外一層查找的屬性的值。

DOM 的 this
