CSS 筆記 - 金魚系列 12 - 側邊選單

發表於 2020-07-23 更新於 2020-08-05 分類於 CSS Disqus：
前後台都會用到的側邊選單，一定要會！

後台或是形象網站一定會有的側邊選單，當然一定要記錄一下。

學習重點

  • flex 教學
  • ::before & ::after 無中生有的僞元素選取器
  • 親代選取器之妹妹選取器與鞭炮串選取器
  • box-sizing 教學
  • relative 相對定位教學
  • absolute 絕對定位教學

HTML 架構

會發現幾乎常見的架構都相當單純。

CSS 樣式

aside 主體

  • 讓 sideMenu 裡面的元素變成橫排
  • 讓元素的方向變成直排
  • padding 因為有 boxsizing 的關係，所以不會有 Y 軸 bar
  • 讓 aside 產生陰影

搜尋列

  • 讓 form 裡面的元素橫向排列
  • 設定 input 與 button 的空間比例，並將兩者的背景做成透明

NAV

  • 設定所有的 fontawesome 圖示都往右邊，超出 nav 範圍，所以使用負值。
  • 使用 scale() 方法，產生位移讓圖示消失。
  • 項目 a 連結後產生一個 a 連結偽元素，使用 + 號選取器選到 a 的所有偽元素都有其樣式，使用絕對定位方式編寫，因為 padding 已設定左右 10px，所以左右邊要推擠 10px，然後使分隔線定位在 top:0px，固定在中間。

CodePen https://codepen.io/hnzxewqw/full/dyGaeeO
