CSS 筆記 - 金魚系列 04 - 交錯漂浮版

發表於 2020-07-14 更新於 2020-07-15 分類於 CSS Disqus：
產品介紹或是公司介紹常會出現的交錯漂浮切版

學習重點

  • 區塊尺寸怎麼算
  • Box-sizing 是甚麼
  • CSS relative 相對定位
  • 網頁的色彩原理
  • flex-direction 的原理
  • nth-child 選取器教學(建議先行閱讀會比較有概念)

非常有趣的主題，立馬來切一下文字與圖片交錯的版型！

HTML 架構

共有四個，會分成文字在左，圖片在右各兩組，其他兩組反之。

CSS 樣式

  • 確認好架構後，在 .item 使用 flex 使之橫向排列，flex 真好用!!讓文字與圖片垂直對齊，讓每* 個區塊下方有 70px 的空間。為了要實現交錯的效果，把圖片與文字的寬度都設定為 55%，這樣相加為 110%，可是 flex 會自動運算空間，所以加上一個這次新學到的屬性 flex-shrink ，使其值為 0，就不會自動收縮。
  • 圖片下方都會出現一個莫名空白，使用 vertical-align: middle;，這個技巧出現很多次，學起來很受用，這樣圖片就不會有莫名空白囉!

然後就可以看到畫面壞掉了，文字超過設定範圍。

子代選取器 :first-child、:nth-child(n) 選取器

  • 為了讓文字可以壓在圖片上，這邊使用子代選取器，選擇第一層第一個物件(看是圖片或是文字)，讓期 margin 為負值，就有交錯的效果。
  • 再來使用 CSS3 新增的 :nth-child(n) 選取器去選擇 DOM 子層元素，不用先命名 class 就可以選到要設定的元素，小括號內的參數可以放順序，也可以放基(odd)、偶數(even)。

    可參考連結有更多說明。

最後在依照想要優化的內容進行優化，交錯飄浮的頁面就完成了!!
