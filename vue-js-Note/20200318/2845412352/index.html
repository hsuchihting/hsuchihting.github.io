el 與 data 的詳述

el

用來把 vue 實體與網頁 DOM 元素做綁定的關鍵實體，當 vue 實體沒有 el 屬性時，就只能透過 vm.$mount()手動編譯。

data

用來存放與實體綁定的資料，當實體建立後，可能過 vm.$data 取得 data 內的資料，實體內則是用 this.xxx 取得。在子元件時，需要用 function() 的形式回傳。

data 內的屬性開頭名稱不可以為 $ 或 _。
