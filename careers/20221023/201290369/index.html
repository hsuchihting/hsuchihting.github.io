進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 24 話

發表於 2022-10-23 分類於 careers Disqus：
低就沒有關係，選擇投資自己最重要

我選擇了昕秀計畫，雖然另一間給的薪水比這個計畫來得好，但我不希望資遣的光景重演，就算比較好的薪水，如果根基不穩固能領多久，不希望又是一個兩三個工作，結果又要找新的機會，

再給自己一個機會，低就沒有關係

其實在電商第一個月與主管晤談後，我心裡就有個譜，技術能力似乎與他們的期待有落差，此時我就詢問了六角學院認識的 T 學長，他建議我當時他任職的公司有培訓計畫，此計畫有兩個階段，總共六個月的時間由公司內部的資深工程師做兩週的前端工程師培訓，兩個半月的內部專案實戰開發，後三個月的外部專案開發，而外部專案開發會與公司的 PM、SA、工程師一起開發，就是直接實戰客戶的專案，完成目標，若兩個階段都通過考核，就會成正式員工。

看到這個計畫非常欣喜，雖然目前還在電商工作，但我想給自己一個保險是沒問題的，而且通知面試時間是在隔年的一月份，所以還需要一點時間，也算是給自己試用期後的一個預備方案，畢竟現在有意願花時間跟薪資，以及人力成本做培訓的公司已經不多，又加上是長達半年的時間做培訓，我看到一個新的契機是在這個機會不用擔心犯錯，而且有問題都會有人回答，這對於轉職的人來說是非常友善的。

投資自己才是長遠的眼光

在被電商公司資遣後，便開始專心預備 T 學長推薦的計畫，細看計畫內容，我參加的是第三屆的計畫，簡單說就是有給薪的培訓，只要具備基本的前端知識都可以報名參加，而我也是透過這個計畫中學習許多開發技巧與了解正規的專案流程。

雖然說有給薪的培訓，但當時對我來說要供應家中經濟其實還是有點吃力，需要精打細算、省吃儉用，但我知道現在的投資對我是最大的效益，把技術掌握好，相信未來薪資會慢慢的成長。

薪資很重要，但不是最重要的

這是這一年第三次找工作時，不再像先前一樣海投，因為已經待過三間不同型態的公司，雖然開發經驗不多，但因為每間公司需求的技術都不同，而且都要直接進專案的壓力下，無形間我在切版與程式邏輯上的成長也變多了，不再像新手麻瓜一樣鴨子聽雷，再加上 Kuro 老師的《0 陷阱！0 誤解！8 天重新認識 JavaScript》一書，讓我在 JS 基礎觀念上打上穩固的根基，所以非常推薦除了網路資源外，也可以多看書，特別是近期鐵人賽有許多不錯的好文章。雖然薪資對我來說很重要，但實務經驗來說，我首要應養成我的程式開發能力，讓我可以在幾年內可以透過大量開發來累積經驗，當時有三間公司在面試，感謝上帝的恩典，最終我得到兩間公司的 offer。

第一年的轉職可以說是屢戰屢敗，而且對我來說每一次的需求跟情境都是全新的，每個需求我都非常緊張跟謹慎的面對，幾乎每晚都睡不好，一直做惡夢，下班騎車跟洗澡還有睡覺，都一直想著明天問題怎麼解，上班騎車也都有點害怕今天會有什麼新需求會出現？每一次我都告訴自己，這可能是我最後一天在這裡工作，但相對的也告訴自己，如果遲早都會離開，我一定要在每次的工作帶點什麼東西走，所以我把我不會的情境用一個我看得懂的方式筆記下來，成為我的資源。

好，扯遠了，最終我選擇了昕秀計畫，雖然另一間給的薪水比這個計畫來得好，但我不希望資遣的光景重演，就算比較好的薪水，如果根基不穩固能領多久，不希望又是一個兩三個工作，結果又要找新的機會，其實這樣一直重新找工作跟面試，身心都很疲憊，家裡張力也會很高，所以在禱告一週後，我選擇了有半年培訓的機會。

現在回頭看，當時的評估與選擇是最適合我的！在這間公司學習到正確的開發流程，也累積前後端溝通的實戰經驗，共同協作開發與版本控制的流程，也有資深工程師願意帶領與分享經驗和 Code Review，學習很多！

不得不說，每次的找新工作都帶著既期待又怕受傷害的心情，因為很擔心是不是又會被資遣，現今回頭看，一切都有上帝的恩典，上帝用最短的時間鍛鍊我的心理建設，讓我一年面對不同的公司、產業以及工作型態。

人心籌算自己的道路，唯耶和華指引他的腳步。箴言 16:9
