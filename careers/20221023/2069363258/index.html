進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 05 話

發表於 2022-10-23 分類於 careers Disqus：
確認方向後該怎麼試水溫才不會被燙死

評估所有可能後，看來只有線上課程比較適合我了，所以決定繼續投入線上課程努力學習，既然已經買了課程，就好好努力想辦法把投資的效果變成最大化吧！於是我重新規劃我學習與工作的時間如何達到兼顧，既然洗頭都洗一半了，總不能把自己嗆死。

因為看到線上課程許多分享文會看到一些標題寫：

「投入兩個月成功轉職前端工程師」
「沒背景、非本科，投入六個月成功轉職前端工程師」

讓我非常心動，原來可以預見六個月後的我，加上去聆聽的實體課程也都表示最長的就是六個月可以轉職。

但那是別人，我可以嗎？

起初，我先綜觀所有的課程章節數，以及先「預定六個月」後我要轉職成網頁設計師的目標前進，在學習 HTML 與 CSS 時進度相當快速，可能是因為寫 CSS，所見及所得非常有成就感，故充滿學習的動力。

但到了 RWD 跟 jQuery 時就開始卡關，此時內心又出現否定自己的聲音。

還好六角學院有學員社團可以提出學習的障礙，減少內心的孤單感，以及聽聽不同背景的人如何調適學習障礙的過程，找出更合適的方法。在社團中才發現跟我同個症頭的同學也不少，所以我非異類，只是剛好走到撞牆期。

規劃學習時間與評估學習效益

從二月買課程到此時已經過了一個月，發現撞牆期的出現非常挫折，但又不知道如何闖過，腦袋思緒很混亂，因為知道自己時間不多，太太五月就要面臨生產，孩子出生一定手忙腳亂，沒時間學習之類的想法因應而生。果然，腦袋混亂，學習進度也混亂，吸收不良導致同個課程開始反覆地看，但卻無法專心學習跟吸收，尤其是在 jQuery 的時候面對編輯器還是會頻頻卡住，可是明明就都學習過了，超級沮喪。而給自己預定的時間又變少了，但學習成效卻不如預期，沮喪又加上挫折。

就這樣低潮學習的狀況過了三天，我發現再這樣下去不是辦法，需要調整節奏與步調，我刻意讓自己停課一天，去觀察與思考我的內心狀態。

此時的學習路徑是 HTML > CSS > RWD > jQuery

時間上也都是沒太固定，中間還有工作要忙，所以都穿插著學習。

後來發現給自己一個固定時間，會安定我自己的內心，少量多餐的效果比較適合我，所以便調整成每天起床時間固定，9:00-10:00 是早上的學習時間，下午則是 13:00-14:00，晚上為 9:00-12:00，分為三個時段，先從這樣開始讓自己重新投入，調整新的步調，發現這樣效果有比較好一點，因為固定時間，所以我內心告訴我，此時段我刻意只專注於學習前端這件事，工作部分我一率先擺著，等學習完後再說，心無旁鶩很重要。

也因為學習成效慢慢變好了，在工作上也不會因為擔心學習進度而分心，工作也比較有效率，內心要走前端工程師的想法越來越明確，但晚上還是會思考，該怎麼爭取更多的學習時間？

改變了最終目標，可是沒有調整時間，因為我忘了一件事，就是我的目標從 「網頁設計師」 變成 「前端工程師」，但現實時間看起來又似乎不太允許我調整。

嘗試的過程總是忐忑不安，徬徨無助，但能做的就是按部就班，步步為營，把眼前的進度都掌握好，邊走邊評估風險，便會看到成果。

他要像一棵樹栽在溪水旁，按時候結果子，葉子也不枯乾。凡他所做的盡都順利。詩篇 1:3
