進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 03 話

發表於 2022-10-23 分類於 careers Disqus：
資料蒐集完了，那怎麼切入新領域？

那是什麼？為什麼是 UI？

當我腦海中有這兩個字的時候，沒有猶豫立即起身做到電腦前 Google，到底 UI 跟未來職涯有什麼關係？其實我知道 UI 就是使用者介面設計，但更想知道為什麼是這個方向，在搜尋許多文章與了解工作內容後，發現好像可行。

投入學習去釐清是否尋求的方向如預期

評估眼下的狀況，加上我了解需要將自己丟入一個環境中，我比較能專心學習，所以在朋友推薦下去上了一位有名的業師的課，價格比我預期的高，但收穫滿滿。經過兩個三個週六的密集課程，發現 UI 也不是我要的方向，如同持續想的那句話：如果我不做設計，我還能做什麼？

UI 整體上很廣義，但在台灣的工作上有點狹義在畫網頁或 APP 的設計稿，實質上還是有點偏設計，加上對於長時間做設計已經有點疲乏，所以經過禱告與思考後，確認 UI 不是我要的方向。

那，我能走去哪？

因為受到太太的影響，她是多媒體設計科系畢業，畢製是拍紀錄片，但對於服裝有熱情，後天努力花了五年學習女裝並投入此領域，最後還考到了甲級女裝技術士證照（猛)！也是看到一個方向是有設計背景為養分，在做可以將美感延伸的工作時，把這素養延伸到新的領域，這比較像是我想要做的。

身為基督徒，我開始禱告，禱告可以讓我專心，此時又觀察到現在手上的工作，近年需要實體印刷的數量越來越少，客戶需要交付的數位化稿件產量越來越多，就想說什麼東西可以變數位化，並且延伸平面設計的經歷？想到網頁不就是這樣的一個平台！

找到方向了，然後呢?

當我上完課與找到方向已經是 2018 年 11 月的事情，發現已經迫在眉睫，不轉不行，故開始找培訓的單位，因為我知道自學我應該會半途而廢，因為我是個容易分心的人。加上太太當時已經有孕在身，預計 2019 年五月臨盆，再不趕快找到一個穩定收入的工作，生產與坐月子的錢哪裡來？

網頁是資訊產業，第一個想到的是資策會，也在說明會的時間去聽，發現費用很高以外，還要花半年時間無法工作，全職投入學習。算一算我需要籌個五十萬才有辦法度過這半年且兼顧生產與月子中心的費用，想想此路不通，但也比較有方向，我要找的是前端工程師，方向有比較收斂後，持續找了其他實體培訓單位，因為我覺得實體培訓我比較有環境可以學習與約束自己，不是申請了沒有通過培訓資格，就是時間與費用無法配合。也有詢問教會的友人是否可以讓我到公司當網頁實習生，但相談後發現我對於網頁知識是零基礎，連基本的術語都聽不懂，怎麼有辦法勝任，可能連實習幫忙都沒辦法。

最後在 Google 上看到了六角學院是線上的培訓單位，且標榜了打破線上課後不理，真人助教批改作業、履歷健檢、模擬面試與求職服務多項的內容。

心想：似乎只剩下線上比較適合我了，時間與家庭上比較可以兼顧，加上課程費用不算高，算是小額投資，先買幾堂課試試看，如果學不來，大不了先回頭做平面設計，騎驢找馬或另做打算吧！

與太太商量後，得到一個結論不管怎麼都要往下走第一步，所以開始找相關的資訊看看哪個比較適合我。

就算前方是未知的道路，好比走在黑色的隧道中，但相信隧道的盡頭會有亮光。

你的話是我腳前的燈，是我路上的光。詩篇 119:105
