進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 14 話

發表於 2022-10-23 分類於 careers Disqus：
過完年了，投履歷測試自己的市場接受度

此時預備的狀態

就這樣學習將近一年，來到 2020 年的 2 月，在沒幾天就要準備迎接農曆春節了，感謝上帝的恩典，雖然 2019 年把大部分的重心都放在前端學習，但每個月還是有夠用的案件來支撐生活開支，雖有高高低低，但整體來說勉強都過得去，也獲得許多親朋好友不時的分享食物與食材，孩子也很體貼地知道老爸賺錢不易，速速的就改吃副食品了(笑)，也感謝太太盡量自己烹煮食物，除了給寶寶有健康的副食品外，也幫家裡節省開銷。

此時也完成了第一個前端靜態作品，使用的技術有：

  • SCSS 手刻樣式與 RWD 響應式設計。
  • 樣式模組化。
  • jQuery 開發互動效果。

雖然說能條列式的技術不多，但能做出靜態頁面網站是可以的。

我抱著嘗試的心態開始丟履歷，打開人力銀行網站，原本覺得看不懂職缺內容越來越明白，覺得真的離轉職不遠了，雖然 JavaScript 還沒有到很熟練， 框架也還沒學習，但感覺還是有沒用框架的公司在徵才，心想：現實考量下，硬著頭皮先踩進這個領域做中學吧！

於是在沒有任何背景下海投 80 間公司，獲得四間的面試機會，雖然都是感謝信或無聲卡，但我很開心的是履歷內容被市場接受了，並且更有目標繼續前進補足不夠的技能與知識。

過去所累積的技能與知識會成為轉職的助力

不斷重提：走過的路絕對沒有白費，老掉牙但是非常真實。

其實我沒有想到會去接觸網頁與程式語言，原本認為是平行世界的程式語言，竟然在多年後與其相處，並且越寫越有趣，甚至變成現在的職業，而過去美術設計的經驗也幫助到我在網頁開發上的美感敏銳度，至少在沒有設計稿與 CSS 框架的情況下，我不會做出很醜的介面，這點與資工系出來的朋友聊過，美感是他們的痛點，也是最難克服的一點，所以過去的經驗累積一定能成為未來的養分，甚至在現在的工作上也可以跟設計人員溝通，因為我跟他們有共同的語言。

過去在美術學習養成對於把想法用口語的方式陳述的訓練，在面試時對我的幫助非常大，加上在教會長大常常需要做分享，用文字及語言把腦海中的想法整理並陳述，是需要一點訓練的，我很感謝過去的自己有累積這樣的能力。

投遞履歷預備的內容

在投遞履歷前，我詢問校長需要準備的資料，很感謝六角學院在履歷健檢這部份給我很多的學習，原來過去準備履歷的方式不是很正確，而履歷也是呈現自我很重要的環節之一，如果不重視履歷，代表也不太重視自己。

準備好一份 CakeResume 履歷

現在有許多履歷的網頁與產品，會推薦 CakeResume 的原因是排版舒服，並且免費的版型已經很夠用，除了網頁版有支援響應式外，也可以下載成 PDF，並且完成都是儲存在雲端並且會產生網址，可以放在 cover letter 讓人資或是面試官在看到人力銀行的履歷前就可以看到客製好的履歷。

也因為資料放雲端，隨時都可以修改內容，就算已經投遞了，可以在半夜修改內容(誤)。而且踏入前端職場後，也可以定期修改履歷，增加曝光度！若履歷建置得不錯，或許會被 CakeResume 相中成為範本履歷喔！

對了，六角學院有額外的履歷健檢服務，如果有註冊任一門課程的學員，就可以享受到這個服務喔！

作品

重質不重量，做一個完整呈現的網頁作品，靜態也行，有串接 API 更好。如果還不會串 API，至少寫個假資料去呈現互動功能。有一個主力的作品，可以再加上幾個 JS 功能的小作品，可以讓作品部分有更多元的呈現。

紙本履歷

雖然網路很發達，但企業中也會有資安規範，不一定都可以看到 CakeResume，故印出紙本還是有其必要性。可以放作品或部落格的 QR code，可讓面試官透過手機掃描瀏覽。

投遞履歷的心態

保持一個開放的心，找工作就跟談戀愛一樣，不要強求，而是要找合適的，得失心不要太重，以免履歷丟出去一直等回音，讓自己成天魂不守舍，心浮氣躁就不好了，收到感謝信與無聲卡也不要太自責，要把履歷跟作品內容在調整，是不是陳述有問題，或是哪邊可以再加強，沒有回應也是一種回應，要用對方立場來看自己的履歷，因高階主管大概能看履歷的時間是五秒，並且如果沒有太突出的特色，會放在一旁不會在瀏覽，除非篩選出來的人數太少，才會在從「移除」的履歷在挑第二輪，故一定要在第一輪被相中，積極會在履歷上的小細節呈現，所以在準備的過程要積極。
