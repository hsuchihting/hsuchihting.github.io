進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 04 話

發表於 2022-10-23 分類於 careers Disqus：
沒有條件了先去試看看吧！狂聽說明會

有一個工作內容會一直進化成長，並且經得起時間的考驗，又可以累積技術，且需要不斷的前進，這跟我內心的工作型態與職涯是相符的。

在開始上線上課程之前，已經先花了一萬多塊並上完 UI 課程後發現不是我想走的路，因為我依然很想跳脫設計的範疇，所以我開始找說明會想進一步的了解，以下是我在 2019 年聽的說明會心得記錄：

資策會

聽這場說明會是 2018 年十月底，這是我內心的第一志願，前往在大安站的大樓去聽說明會，到了櫃檯拿了一本厚厚的講義，以及進到一間很多電腦的教室，來的人大概只坐滿教室的四分之一，可惜講了一堆我有聽沒有懂的電腦科學專有名詞，但介紹的老師提到，許多的東西都在退步，但資訊一直再進步，而且不會往後退。

就是這個！

有一個工作內容會一直進化成長，並且經得起時間的考驗，又可以累積技術，且需要不斷的前進，這跟我內心的工作型態與職涯是相符的，當下聽完心想：「就是這個了」，而白板上寫著三個職務：

  1. 網頁設計師
  2. 前端工程師
  3. 後端工程師

為了跳脫原本的平面設計領域，我想前端工程師應該是最適合我了，其實當時的選擇很好笑，只是界在三個職位的中間而已，但聽到學習時間需要全職學習半年加上就算聽完說明會後的學費還要十萬，面臨家庭狀況這條路我知道不通！

但至少知道前端工程師是我要的，而此時我也購入了六角學院的網頁設計師課程。咦？不是要走前端嗎？怎麼會先買網頁設計師課程，因為我對自己沒有太大的信心，要從圖像式的畫面走入密密麻麻的程式碼領域，多少會怕，想說先試著水溫，若學不成至少財損不要太大。畢竟原本光是 UI 的課程已經花了不少錢…

AppWork School

對於前端工程師的工作環境與內容越了解後，就越來越嚮往了，發展性高，成長幅度也有，技術端的深入研究或是未來管理職的走向，都可以有一塊很大的藍海可以前進，而且面對的不是只有台灣，而是全世界。

契機

此時為 2019 年二月底，已開始六角學院的學習，而學習的這段時間，也看到許多一起在前端奮鬥的夥伴們，雖然彼此不認識，但有一個問題拋出來，就很多學長姐跟厲害的前輩會指點迷津與討論，也因為這樣認識了一位大學同校的 T 學弟，雖然差了六屆，但因為有一層校友關係，拉進了一點距離！感謝他介紹前端資源給我，也特別感謝他某天的下班來陪我吃飯並分享轉職要注意的要點，其中一個就是這個學習機構 AppWork School。今天是他們公司的員工出來分享，透過 AppWork School 的課程，如何轉職成工程師。其中分享的講員有分以下三個領域

  • iOS develope
  • Front-end Engineer
  • Android develope

不同領域，不同看見

各別工作時間為兩個月到兩年多，但因為我明確的方向是網頁前端開發，所以我就紀錄一下網頁前端開發這部份。

分享者是一位年輕的女孩，桃園人，大學科系是行銷，進入職場後發現幾個問題後，最主要的是對於行銷工作已無成長空間，雖然中間遞出辭呈後，被人資調往其他部門，但仍然是做行銷。最終她在一年後還是選擇離職，後進入 AppWork School。

聽她分享中間還是會聽到，本身對於程式有一點興趣，所以有去接觸相關課程，或是自學！果然會分享到面臨自律與學習障礙的高牆。

分享會後有一個簡短的 FAQ，我也提出一些我想問的問題。但礙於時間又不想奪去其他人的權益，在會後講員們也很熱心的留下來回答問題。從她身上我詢問到幾個我想問的重點。

  1. 在 AppWork School 是用專案的方式來進行教學與學習，早上九點到晚上六點，其中早上的時間是彼此溝通專案內容與如何完成，下午就是寫程式呈現畫面的時間。晚上六點後可自主留下來練習與討論，但基本上是在個人技術上的練習，因她住桃園，六點就收拾包包回家去了。若是在個人技術端的練習，其實我也會選擇回家進行，畢竟小孩剛出生，還是需要兩個人一起照顧，也降低太太的負擔。
  2. 有問到測試瀏覽器這部份，她回答在學習中不用買 Macbook，因為當中會有 iOS 開發學習者，所以可以借他們的電腦來測試相容性。這也讓我安了心，畢竟要投資一台 MacBook 也是要不少的預算。
  3. 她推薦我先把 javascript 的部份先學起來， jQuery 基本上不用太理它，業界較少人用，幾乎都用 javascript 來寫，不過如果 libary 可以使用，我是不排斥，但 javascript 是前端必備就正面對決吧！
  4. 她表示課程中一開始是完全手刻 RWD，不使用框架，為了奠定更穩固的 HTML & CSS 的原理與概念，這很 OK！
  5. 最令我心動的地方是程式碼卡關時，馬上有人可以問，有一位資深技術導師會協助學員！這是我考量的要點之一。
  6. 前端工程師在發問前要有獨力解決問題的能力，所以 Google 大神是首選。其他就是閱讀官方文件。
  7. 最後她也表示，其實業界對於 Junior 工程師的要求沒有到太高，但要有一顆熱忱與學習的心，讓公司覺得可以投資在自己身上，這點超重要！

更加確定前端工程師這條路

不管如何，我將這件事情放在禱告當中，求上帝帶領，總共需要花 20 週的時間學習，透過此課程成為前端工程師的人目前在一萬多人中有 89% 的就業率，是算滿高的，但駐點練習需要全職投入的決心，這點非常挑戰，現在有妻小，真的不是要投入就投入…

雖然分享的畢業生口條不完全算很好，但看到她們從這邊結業後進入職場，那充滿自信與笑看這一切努力的過程，還是令人相當羨慕的。

這位前端說，進入業界雖然有高階工程師可以問，但絕大部分還是要自己解決問題，我給自己一個期待與目標。

後來去申請收到一份全英文的 Google Form，填寫完後就幾天只收到感謝信，進而了解得知此單位看不出我有強烈企圖想轉職的動機，雖然確切原因還是不知道是為什麼，但可以知道的是這條路也沒有了！

五倍紅寶石 DEMO DAY

剛好在 ACCUPASS 看到五倍紅寶石的 DEMO DAY 的免費活動，想說看一下人家 DEMO 內容都在幹嘛，就直接點下報名，然後當天活動開始就去了，此時已經累積一點基本的切版能力，至少到現場不會像鴨子聽雷，跟麻瓜一樣坐在台下浪費時間。身邊來聽的不乏也有資工或資管系的學生，與他們會前簡單交流後發現原來就算本科系也是很多不太會寫程式的，這讓我稍微喘口氣。

而開始聽各組報告他們這幾個月從零到有規畫主題、需求確認、網站前後端開發、專案控管…等等的，在如此短的時間竟然可以開發一個完整的網站，讓我眼睛為之一亮，會後交流與五倍人員了解後，依然需要全職投入學習，此選項實在無法配合，但內心再次篤定前端工程師確實是我想要發展的職涯了！

縱使眼前看起來不如預期，但要相信一切所經歷過的人事物，都有上帝最美好的安排。

我們曉得萬事都互相效力，叫愛上帝的人得益處，就是按他旨意被召的人。羅馬書 8:28
