進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 19 話

發表於 2022-10-23 分類於 careers Disqus：
被資遣了！要喜樂並抬頭挺胸的走出去

慘遭滑鐵盧，每天都很想離開

前篇說得意氣風發，馬上這篇來打自己臉，帶著期待又愉快的心情來這間公司報到，位在南港軟體園區內，日系管理風格，寬敞的辦公室，並且用自己同仁所開發的打卡程式，手機打卡上下班，公司制服是商務休閒，整個人上班就好像一個商務人士，配著人生第一台的蘋果電腦，讓我很享受這次的報到（笑)，果然菜鳥就是菜鳥。

加入的團隊是做提供製造業工作流程系統的產品，使用的是 Angular 框架開發，對當時的我來說，完全沒有框架經驗，物件陣列的操作也沒有很熟悉，就只是會簡單的切版跟使用 jQuery 控制一點網頁元素而已，當時連串 API 的經驗都沒有。

直接上戰場然後變成砲灰

雖說會給時間學習跟養成，工作有提到會需要畫 UI，因我是平面設計背景轉職，學習 Adobe Xd 上手的時間算快速，從學習到畫出 mockup 大概只花了三天時間，並且交付稿件給主管去 demo，這邊的案件時間都滿趕的，所以張力也滿大。常常聽到隔壁的後端同事會喘大氣的聲音。

因為是使用 Angular，主管安排一位資深工程師帶我認識此框架，大概花了三個小時講解整個 Angular 的生態跟架構介紹，但我聽得懵懵懂懂沒有很理解。後又安排另一位工程師出作業給我。這時我才知道學習 Angualr 也會需要同步學習 TypeScript 還有 RxJS 的概念，JavaScript 還沒有很多實戰經驗的我，整個非常慌張，而第一份作業是要做一個登入表單並且要完成驗證功能，驗證通過後要用路由跳轉頁面，並作路由切換 SPA 頁面。

在整個基礎知識都不穩固的狀態下，上網找資料勉強拼拼湊湊出一個看似有類似功能的登入表單以及 SPA 的目錄頁面，但問題一定是一堆，而且基本觀念不好的狀況下，被電得體無完膚。

因為前端只有兩位，都要從 UI 設計產出到開發接一人作業，故幾乎沒什麼時間理我的提問，大部分的時間都回答：「這個可以先 Google 跟看一下官方文件，有問題在問我。」雖然現在回頭看覺得是正常，可是這跟一開始面試所表達會有培訓跟我認知的培訓方式是有落差的。

五天開發一個打卡系統

某天開會主管聽完其他工程師對我的想法，對我說：「聽起來已經給兩個多禮拜學習框架，但似乎成長有限，是否要思考要繼續往寫程式的路呢？其實不是每個人都適合寫程式的。」聽完我真的內心受到很大的打擊。

最後給我一個終極考核，就是在三天內要仿照現在的打卡系統，並於三天的工作天做完，要已能進入專案開發為目標。需求如下：

  1. 切版需要符合 RWD。
  2. 直接參考現行打卡網站，不提供設計稿。
  3. 使用 Angular 開發。
  4. 串接 API。
  5. 三個工作天完成。

當時開會後的時間是週五下午四點半，回到座位後，後端工程師就給我一份 API 的文件，告訴我那些功能要呼叫對應的 API，剩下的前端部分就要靠我自己了，算一算時間因為只有三天，所以會在週三的下班前要 demo 給主管看，週五回到家告知太太此事，太太便二話不說力挺，晚餐開始到睡覺都不吵我，小孩一手照顧料理，我匆匆吃完晚飯便坐在電腦前開始開發。

孩子睡了，她走到我身邊說：「如果給你時間這個專案你做得完嗎？」我想了一下說：「可以，但真的需要給我多一點的時間。」

「那就跟它拚一次看看，盡力而為，把自己能做的擺上，剩下我們禱告上帝。」當下我內心充滿了力量，我想盡辦法在我當時的能力內做到我可以做的。

在週末也上網找很多的資料，也很感謝一位六角學院的 M 學姊願意在週六的下午與晚上陪我解問題。

週一上班主管臨時告知，週三因為下午有場會議，故 demo 時間提前到早上 11:00，這下可好，我能開發的時間瞬間少半天，加上公司還有安排原訂的工作業務給我做，那幾天我每天都寫到早上五點，去小睡一下七點半又起床盥洗上班，非常痛苦的五天。當然以當時的能力是小孩開大車，專案無法如主管預期，功能也不完善，最後就以資遣收場。

不是盡力，是一定要做到

雖然最後沒有通過考核，但主管最後表示他覺得滿感動的一點是，我沒有找藉口，沒有說因為工作有其他事務所以耽擱進度，還是看到我有在努力，但可惜這邊似乎不適合我，但沒有完成就是沒有完成，說再多也是多餘。

這次經驗再次告訴我，有熱情跟多努力是不能表達什麼的，成敗論英雄，結果論就是這個社會的才可以接受的。我將此是同步分享給我太太以及教會的牧師，牧師給我鼓勵與安慰，縱使心裡難過，但我調整心態，若我技術能力夠，我一定待得下來，至少我更明確知道職場需要的工程師能力要到什麼門檻，於是我收拾我的情緒，改變內心狀態，一樣待到下班歸還設備，並且跟部門的每一位同仁道謝，感謝他們這段時間的照顧與提點，最後工程師們也給我一點學習的路徑跟方向，我便在這趟旅程撇下一個逗點，給自己喘口氣後繼續挑戰下一站。

很感謝六角學院的校長－洧杰在當天晚上九點半還特別致電給我，關心我的狀況，也提點我一些要補強的地方，電話那頭聽得出來正在搭捷運，校長表示他剛結束晚間的教學正在返家的路上，沒有數落、沒有責難，是平和的語氣跟中肯的建議，非常感謝這段旅程的陪伴，相當感動！

或許轉職了，會覺得我已經完成設定的里程碑，但當鬆懈時也就是危機的開始，反正事實已經發生，時間與機會是不會因為自身情緒難受或低落而停止，明天早上都會是新的，難過一天就夠了，要趕緊振作；開心也只要一天就夠了，奇怪的需求馬上就會出現。轉職第一年有陣痛期，可長可短，但不管如何，更重要的是如何再次回到這個職場，所以沒有停下腳步的空間啊！

弟兄們，我不是以為自己已經得著了；我只有一件事，就是忘記背後，努力面前的，向著標竿直跑，要得上帝在基督耶穌裏從上面召我來得的獎賞。腓立比書 3:13-14
