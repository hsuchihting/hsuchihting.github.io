進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 11 話

發表於 2022-10-23 分類於 careers Disqus：
啟動！用部落格寫下自己的足跡

在八月份的時候有看到社團在鼓勵參加 iT 鐵人賽，這是在投入前端領域之前不知道的事情，但看到報名頁面超級多人報名，規則是連續三十天寫技術文章不間斷，而且中間會遇到中秋與國慶連假，不外乎是一個不容易的挑戰，不要說撰寫的內容要多有深度，光是完賽就不是件易事。

選擇任何可以讓自己變強的管道

為了讓自己變強，我選擇目前比較熟悉的切板來當作鐵人賽的系列，第一次參加不知道是否可以符合大賽主題類別，又擔心寫得不好，故選擇自我挑戰組，強迫自己每天練習切版用，希望可以透過外在的力量把自己再往前推一點，但因第一次參加，我真的每天寫一篇，也就是每天切一個版，真的超硬，每天都很緊張怕自己錯過或斷賽，甚至有一天電腦還故障，到當天晚上 10:30 才排除障礙，還好有趕上當天鐵人賽發文，後來才知道可以先累積文章在參賽，壓力會比較小(哈)，但也透過這次的鐵人賽練習，分了四個主題來切版，間接在這次鐵人賽累積不少切版實作經驗，在切版上也越來越熟練，SCSS 以及 CSS 屬性也掌握較多的知識與能力。在這個過程中雖然緊湊又緊張，但切版的那端時間，滿享受沉浸在寫程式的世界，思緒很連貫，且完成畫面後成就感很高，是一個自我對話的過程，很像在大學時做藝術創作的感覺。

故接下來的幾年我都有參加鐵人賽，並且透過鐵人賽找一個想學的主題來撰寫，透過系統式的學習，讓我可以更了解相關技術的內容。

寫鐵人賽最大的獲益是自己

終於經過中秋與國慶連假的挑戰，我完成了第一次的鐵人賽完賽里程碑，這過程真的很享受，也看見自己的成長，看到文章有被肯定還有留言，是莫大的鼓勵，原來自己的東西真的有人看，且有實質幫助到人，並且有資深的工程師樂意提出更好的解法，怎麼說都是賺到。也透過鐵人賽為自己留下一點成長的軌跡，再回頭看都是滿滿的恩典與感動。透過鐵人賽也逐漸結交到一些前端領域的朋友，視野也逐漸擴大，眼界也不再局限於現在的困難與挑戰，很感恩的是去年的鐵人賽還被受邀出版成書籍，算是意外的收穫。目前在天瓏書局網站販售中。

購書傳送門：https://www.tenlong.com.tw/products/9786267146460?list_name=i-r-zh_tw

適度的壓力是成長的最佳養分

我是需要一點外在壓力讓自己成長的人，自己獨跑比較容易放棄，畢竟不努力比較輕鬆嘛…但努力流點汗，讓身體感到疲勞，精神受點壓迫，完成後的被成就感填滿的瞬間，真的非常滿足，會想繼續挑戰這種感覺，最重要的可以把所學的知識內化，可以用自己表達方式說出一套讓想學習以及非本科的朋友了解，才是真正有學習到這門技術的要領。
