進擊的轉職-平面設計邁向前端工程師之奇幻旅程 第 23 話

發表於 2022-10-23 分類於 careers Disqus：
原來養成也是會養不成的，是否還要走這條路？

原來養成就是不斷的考核？

進入之後大概前幾天是公司環境與工作介紹與了解、開發環境設定，以及程式碼研讀。後來會安排案件給我實作，大概都是三到五天的程式改寫，使用技術是 Lavaral + Vue，前端部份是要改寫成 Vue，我用一個一知半解的 Vue 概念去改寫每個畫面，但當遇到陣列物件的資料整理，就會卡關很久，此時發現進入前端框架如果陣列物件的操作不熟練，會一直撞牆，中間有跟校長諮詢，他表示我目前都在同樣的問題打轉，應該要趕緊把陣列物件的方法掌握起來，不然一樣的狀況會一直重演。

雖然說可以詢問其他工程師，但幾乎每次的任務都要跟工程師提問，才能在 deadline 完成開發，就這樣連續考核了五週。

每一週我壓力都非常大，因為我感到吃力，先前的焦慮感一直不斷的上演，每天我都睡不好，而且低潮感越來越明顯，雖然我還是很努力地打起精神讓自己天天開心，去享受工作的片刻，但內心的壓力跟開始嗅到可能要被資遣的氣息。

這邊還是很感謝幫忙協助解惑的工程師，M 同事、P 同事跟 Y 同事，都是非常資深的工程師，並且前後端都專精，非常厲害。

最終考核奮力一搏

在第七週主管跟資深工程師給我了一個不一樣的任務，這個任務不是主要專案內的，而是去連一個 open data API 並要按照需求完成三個頁面的功能。此刻我心裡知道，這是我最後一個考核，如果沒過一定又是資遣收場了，工程師講解完後，主管表示這個任務我多久可以做完，我表示大概要一週。但他覺得太久了，如果七天應該要再加更多功能，所以他評估完說應該五天要可以完成，不過這次他想挑戰我的極限，壓縮成三天，並且不能 call help。

這次一樣又是跨週末的測試，雖然有上一次的經驗，但內心依然緊張，我知道靠自己目前的程度一定是無法通過，為了不要歷史重演，這次我還是私下請求教會的工程師朋友，他是一位資深工程師，我都稱他為師傅，以下稱為 K 師傅，很驚喜的他願意協助我走過這次的考核，所以我週六到週一的晚上花三個小時請他指導我串接 API 的方法，我能做的就是先把切版完成，並且串接假資料，第一個晚上他教我如何串接基本的 API 時，我覺得人生充滿了亮光，就像小當家吃到美食一樣的亮光！

幾乎完成了 70% 內容，也透過這次的考核讓我把比較不熟的陣列物件的操作方法學習起來，雖然高壓，但跟著 K 師傅一起開發，學習非常多，思考問題與需求的方向也跟以前不同，真的非常感謝他。

我盡力了但最終還是養不成

「不是盡力，是一定要做到」這是九把刀的一本散文集的封面標題，在當兵看完這本書也有滿多感觸，職場社會看重的是結果，過程當然也很重要，但最終的目標就是希望得到如預期的結果，雖然我最終得到的考核成績是 80 分，但主管覺得需要 call help 這件事情，在這間公司可能還是沒辦法協助我，這次最終考核有點像是給我設定一個 side project，透過這個作品讓我去學習前端框架、API 串接、陣列物件操作方法、第三方套件使用等技術，不得不說的確透過這次的練習成長很多，也是有點奠定程式開發基礎，眼界也被打開許多。

於是在平安夜這天，我又被資遣了，這對一個喜歡平安夜的基督徒來說，實在衝擊太大了。我第一時間跟太太說，她倒抽一口氣，可是沒有責難我，反而當下還鼓勵我，但我自己很沮喪。回到家她就爆哭了，因為她比我更沮喪，覺得都這麼努力了，為什麼還是沒有過試用期，這兩場資遣的衝擊，的確深深的打擊我，也開始懷疑是否要走這條道路，上帝啊！這真的是祢要我走的方向嗎？

聆聽內心的聲音並且禱告

也因為靠近年底了，整個社會工作的步調明顯的緩慢下來，而在農曆年前也沒什麼機會面試了，這天晚上很感謝教會的牧師以及幾位好朋友邀請我們全家聚餐，並且請我們吃頓好的，就如聖經詩篇說：在我敵人面前，你為我擺設筵席，你用油膏了我的頭，使我的福杯滿溢。牧師跟我說：「今年辛苦了，看得出來你很努力，但不要擔心，好事還在後頭，我們先舉杯慶祝迎接新的開始。」其實我非常感動，因為打給我的時候有提到晚上想找我們吃飯，但如果我們不去這場飯局就取消，原本不疑有他，想說是頓單純的飯局，但沒想到是一個慶祝新開始的飯局，真的非常感謝牧師與一起聚餐的弟兄一家。

晚上睡前依然跟太太一起禱告，這次內心比較沒那麼難過了（可能吃飽喝足的關係），將這一年的心情向上帝傾訴，感覺很舒服也暢快，雖然父親離開了我，好像失去了一座山可以倚靠，但還好我還有信仰、還有家人、還有教會的陪伴，讓我可以繼續打起精神挑戰前端之路！

一年換到第三間公司，否定自己的聲音越來越高，也會懷疑自己是否還要走這條路？但我抓住我禱告的領受，我相信上帝的安排跟引導一定沒有錯，或許這些滑鐵爐是要先訓練我建立比較正確的心理建設，有正確的心理建設或許在面對解題時比較不會腦袋打結，畢竟如果目前面對的是前端工程師的日常，那我應該要調整的是心態，而不是追求答案。

因我們行事為人是憑著信心，不是憑著眼見。哥林多後書 5:7
